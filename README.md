# 🔐 Password Generator

A clean, modern password generator tool built with vanilla HTML, CSS, and JavaScript. Generate strong, secure passwords instantly — no backend, no tracking, no ads.

![Password Generator Preview](./assets/preview.png)

## ✨ Features

- 🎚️ **Adjustable Length** — Slider from 1 to 20 characters
- 🔠 **Uppercase Letters** toggle
- 🔡 **Lowercase Letters** toggle
- 🔢 **Numbers** toggle
- **Symbols** toggle
- 💪 **Strength Indicator** — Red / Yellow / Green visual feedback
- 📋 **One-click Copy** — Copy password to clipboard instantly
- 🔀 **Fisher-Yates Shuffle** — Cryptographically fair password randomization

## 🛠️ Tech Stack

- HTML5
- CSS3 (Custom Properties / CSS Variables)
- Vanilla JavaScript (DOM, Clipboard API)
- Google Fonts — [League Spartan](https://fonts.google.com/specimen/League+Spartan)

## ⚙️ How It Works

1. Select desired password length using the slider
2. Check the character types you want included
3. Click **Generate Password**
4. Click the copy icon to copy to clipboard
5. Strength indicator updates automatically based on your selections
